# Employee Directory

A responsive React app that displays employees fetched from https://dummyjson.com/users. It includes search, sorting, pagination, favorites saved in local storage, a dark/light theme, and mobile-friendly behavior.

## Setup

1. Install dependencies:
   npm install

2. Run the app:
   npm run dev

3. Open the URL shown in the terminal (usually http://localhost:5173)

## Features
- Fetches employee data from a public API
- Search by first name, last name, or email
- Sort employees A-Z or Z-A
- Pagination with 10 employees per page
- Favorite employees saved in local storage
- Dark and light mode toggle
- Responsive layout for desktop, tablet, and mobile
- Mobile-friendly installable web app experience

## Build

To create a production build:

npm run build
